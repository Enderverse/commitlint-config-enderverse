## Install

npm
```sh
npm install Enderverse/commitlint-config-enderverse --save-dev
```

yarn
```sh
yarn add Enderverse/commitlint-config-enderverse -D
```


## Usage

Add the following to your `.commitlintrc` or `.commitlintrc.json`:
```json
"extends": [ "enderverse" ]
```
